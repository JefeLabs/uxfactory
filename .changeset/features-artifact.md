Features artifact registered end-to-end (`.uxfactory/artifacts/features.json`).
Resolver-consumed class: it appears in no requires block — it never gates,
only scopes (coverage denominator, generation scoping, extend unit). The
interview elicits capability names ([E]) with status defaulted to planned
([F]); story assignment and inherited/net-new origin are derived — assignment
clusters the registered stories set (hard prereq `features ← stories`), origin
comes from the project quadrant. Bridge reports the row; the panel offers the
create affordance with prerequisite chaining.
