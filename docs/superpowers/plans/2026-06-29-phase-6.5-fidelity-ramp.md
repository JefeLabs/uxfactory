# Phase 6.5 ‚Äî Four-Dial Render Scope + Rubric + High-Fidelity Renderer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scope a batch by a four-dial **render scope** `{visual, editorial, coverage, flow}` so each gate binds only when the scope meets its per-dial thresholds, gating the rubric the SKILL.md loop judges against.

**Architecture:** A pure `src/batch/scope.ts` owns the `Level` ordinal (`none<low<medium<high`), the `RenderScope` vector, presets, per-gate `GATE_THRESHOLDS`, the `binds()` rule, the required-input manifest, and `checkReadiness`. `runBatch` becomes scope-scoped (binding gates run; non-binding ‚Üí `not-owed`; unbuilt tiers ‚Üí `declared`). `batchCmd` resolves the scope (registry base + flag overrides), enforces readiness (missing REQUESTED artifact ‚Üí exit 2 with a structured list), and renders previews via a renderer chosen by the `visual` dial (resvg at `low`, Playwright at `‚â•medium`, graceful fallback). The batch SKILL.md drives the loop. Self-contained; no in-engine LLM.

**Tech Stack:** Node `>=20.10`, TS 6.0.3, ESM/NodeNext, `.js` imports, `verbatimModuleSyntax`. Extends `@uxfactory/cli` (reuses the existing four gates, `loadSpec`, `specToSvg`, `svgToPng`, `EXIT`, `IO`, `BridgeClient`). `playwright` optional devDep (lazy). Vitest 4.1.9.

## Global Constraints

- **The authoritative spec is `docs/superpowers/specs/2026-06-29-fidelity-ramp-design.md`** (the render-scope design) + companion PRD `.plans/UXFactory-Design-Artifacts-and-Models-PRD.md` ¬ß6.5. Implement its tables verbatim.
- WORK ON `main`. Engine SELF-CONTAINED ‚Äî no external cloud/runtime refs, no in-engine LLM (the loop lives in the SKILL.md).
- **Render scope is FOUR dials. A gate binds iff `scope.<dial> >= check.min_<dial>` for ALL of {visual, editorial, coverage, flow}.** No scalar level; no one-gate-one-dimension rule.
- `none` is a check-threshold value only; scope dials are `low|medium|high`.
- REUSE the existing gates (ids: `requirement-coverage`, `reuse`, `coverage-orphans`, `token-conformance`, `flow-reachability`); never duplicate them.
- paths only in tsconfig.typecheck.json; `@types/node` devDep; built artifact verified; scoped commits (never `git add -A`).
- Playwright OPTIONAL: lazy-import; unavailable at `visual‚â•medium` ‚Üí fall back to resvg + a `declared` note (never hard-error); the live render test SKIPS when the browser is unavailable (hermetic suite never depends on it).

**Interfaces (per the design doc ¬ß2‚Äì¬ß5):**

```ts
export type Level = "none" | "low" | "medium" | "high";
export const LEVEL_ORD: Record<Level, number>; // none0 low1 medium2 high3
export type Dial = "visual" | "editorial" | "coverage" | "flow";
export type RenderScope = { visual: Level; editorial: Level; coverage: Level; flow: Level }; // dials low|medium|high
export type GateThresholds = {
  min_visual: Level;
  min_editorial: Level;
  min_coverage: Level;
  min_flow: Level;
};
export const PRESETS: Record<
  "wireframe" | "content" | "visual" | "interactive" | "production",
  RenderScope
>;
export function parseScope(
  input: string | Record<string, unknown>,
): { ok: true; scope: RenderScope } | { ok: false; message: string };
export function resolveScope(
  base: string | object | undefined,
  overrides: Partial<Record<Dial, Level>>,
): RenderScope | null;
export const GATE_THRESHOLDS: Record<string, GateThresholds>;
export function binds(t: GateThresholds, s: RenderScope): boolean;
export function bindingGateIds(s: RenderScope): string[];
export function requiredInputs(s: RenderScope): string[]; // mandatory input of a binding gate
export function declaredFuture(s: RenderScope): { artifact: string; dial: Dial; level: Level }[];
export function checkReadiness(
  s: RenderScope,
  present: { stories: boolean; tokens: boolean; flow: boolean; specs: boolean },
): {
  ready: boolean;
  missing: { artifact: string; dial: Dial; level: Level; action: "provide-or-generate" }[];
  declared: { artifact: string; dial: Dial; level: Level }[];
};
```

**Values (from the design doc tables):** PRESETS ‚Äî wireframe `(low,low,low,low)`, content `(low,high,medium,low)`, visual `(high,medium,medium,medium)`, interactive `(high,high,high,high)`, production `(high,high,high,high)`. GATE_THRESHOLDS ‚Äî `requirement-coverage`/`reuse`/`coverage-orphans` = `{min_coverage: low, others none}`; `token-conformance` = `{min_visual: medium, others none}`; `flow-reachability` = `{min_flow: medium, others none}`. requiredInputs: `requirement-coverage` binding ‚Üí `stories`; `token-conformance` binding ‚Üí `tokens`; `flow-reachability` binding ‚Üí `flow` (reuse input optional ‚Üí never required). `binds` compares via LEVEL_ORD on all four dials.
